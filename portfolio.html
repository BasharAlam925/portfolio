Projects

    COVID-19 Vaccination Sentiment Analysis: Insights from Twitter Data

    This project aimed to analyze the sentiment surrounding COVID-19 vaccination through Twitter data. By leveraging Natural Language Processing (NLP) techniques, I was able to gain insights into public sentiment towards the vaccination process during the pandemic. Here's how the project was structured:

    Key Steps and Techniques:

    • Text Preprocessing: Used advanced text preprocessing techniques such as tokenization, stop-word removal, and n-gram analysis. These steps were essential to clean and prepare the text data for modeling, ensuring high data quality.
    • Feature Extraction: Employed CountVectorization and TF-IDF Vectorization to convert text data into high-dimensional feature vectors. These techniques helped capture the most significant words and phrases that contributed to the sentiment.
    • Sentiment Classification: Built a robust sentiment analysis pipeline to classify tweets into three categories: positive, neutral, and negative. The pipeline was optimized using several machine learning algorithms.
    • Machine Learning Models: Engineered and systematically evaluated a range of machine learning models, including: Decision Tree Classifier, Logistic Regression, Random Forest Classifier, Support Vector Machine, and more.
    • Model Evaluation: Used cross-validation to assess model performance and selected the best-performing model based on evaluation metrics like accuracy, precision, recall, and F1-score.

    Outcomes and Insights:

    - The project successfully identified public sentiment trends toward the COVID-19 vaccination, revealing key insights such as:

    • - Higher positive sentiment in regions with strong vaccination campaigns.
    • - Neutral sentiment in areas with mixed opinions about vaccination safety.
    • - Negative sentiment primarily driven by concerns about vaccine side effects and misinformation.

    This project not only demonstrated my expertise in text analytics and machine learning but also provided a timely and valuable analysis of public sentiment during a global health crisis.
